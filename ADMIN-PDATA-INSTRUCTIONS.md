# Admin PData and Equipment Update

## Admin tabs

Open `/admin/` and select **Workspace**. The five tabs are Main, PData, For_PEPCO_Wrkbk, For_PEPCO_Online, and Invoice.

## PData

The PData tab contains 47 supplied rows and 18 columns. Select a cell to edit it. Use **Save changes** to store edits in the current browser. You can add, duplicate, delete, search, reset, or export the data as CSV.

The auditor app uses the built-in PData values as Proposed Device suggestions. If the administrator and auditor app are used in the same browser profile, saved PData edits are also visible to that app. Cross-device PData synchronization will require a future Supabase table.

## Equipment

Interior and Exterior Equipment both provide HVAC and Lighting modes. Lighting lines include existing-device details, image, Proposed Device, Proposed Qty, Ctrl#, and Notes.
