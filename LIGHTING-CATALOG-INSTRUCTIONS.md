# Lighting Catalog 2.3.1

The lighting inventory now contains the full category and device-code list supplied on July 23, 2026.

## Behavior

1. Select an Existing Device Category.
2. The Existing Device Code dropdown displays only codes mapped to that category.
3. Select **Other / enter manually** to type a code that is not listed.
4. Categories without supplied codes automatically show the manual-code field.

## Categories without supplied codes

- Five Foot Fluorescent T8
- Neon

These categories remain usable through manual entry.
